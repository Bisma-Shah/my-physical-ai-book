# Hardware Requirements

This course requires high-performance hardware due to simulation, perception, and AI loads.

| Component       | Recommended                              | Purpose                              |
|-----------------|------------------------------------------|--------------------------------------|
| GPU             | NVIDIA RTX 4070 Ti or higher (12GB+ VRAM)| Isaac Sim and rendering              |
| CPU             | Intel i7 (13th Gen+) or AMD Ryzen 9      | Physics calculations                 |
| RAM             | 64 GB                                    | Complex environments                 |
| OS              | Ubuntu 22.04 LTS                         | Native ROS 2 support                 |
| Edge Device     | NVIDIA Jetson Orin Nano                  | Real-world deployment                |
| Sensors         | Intel RealSense D435i                    | Depth vision and IMU                 |

Cloud alternative: AWS g5 instances for simulation.
